# webserv
This project is about writing HTTP server. You will be able to test it with an actual browser.

## How to build server
```
make
```

## How to run server
./webserv <port>
```
./webserv 8080
```

## How to build client
```
make test
```

## How to run client
./client <host> <port>
```
./client ::1 8080
```
